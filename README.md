# Flutter Calculator App

## Description
A simple Flutter calculator app that performs basic arithmetic operations like addition, subtraction, multiplication, and division.

## Features
- Clean and responsive UI
- Handles all basic operations
- Supports decimal input
- 'AC' button to clear the screen
